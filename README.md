# 433Mhz

## Installation

1. ```sudo apt-get install wiringpi```

2. ```sudo apt-get install git```

3. ```git clone https://github.com/skytylon/433Mhz.git```

4. ```cd 433Mhz```

5. ```make```

## Utilisation

### Envoyer un code
```sudo ./codesend CODE```

### Recevoir un code
```sudo ./RFSniffer```
